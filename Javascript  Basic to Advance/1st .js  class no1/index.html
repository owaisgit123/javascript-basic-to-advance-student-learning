Data Types in

Javascript

There are two types of data types in JavaScript

  1. Primitive
  2. Non-Primitive

Primitive Data Types

  • Number
  • String
  • Boolean
  • Null
  • Undefined
  • Symbol

Non-Primitive Data Types

  1. Object
  2. Array
  3. Function

Type Checking in JavaScript

You can check the type of a variable using the `typeof` operator.

JavaScript String Properties & Methods (Notes)

Property

  • length: Returns the total number of characters in a string.
    "Hello".length // 5

Methods (Without Arguments)

  • toUpperCase(): Converts all characters to uppercase.
    "hello".toUpperCase() // "HELLO"
  • toLowerCase(): Converts all characters to lowercase.
    "HELLO".toLowerCase() // "hello"
  • trim(): Removes whitespace from both ends.
    " hello ".trim() // "hello"

Methods (With Arguments)

  • includes(substring): Checks if substring exists. Returns true/false.
    "Hello World".includes("World") // true
  • indexOf(substring): Returns index of substring, -1 if not found.
    "Hello World".indexOf("World") // 6
  • replace(search, replacement): Replaces first occurrence.
    "Hello World".replace("World", "JS") // "Hello JS"
  • replaceAll(search, replacement): Replaces all occurrences.
    "Hello Hello".replaceAll("Hello", "Hi") // "Hi Hi"

Other Useful Methods

  • concat(): Joins two or more strings.
    "Hello".concat(" World") // "Hello World"
  • padStart(targetLength, padString): Pads start of string.
    "5".padStart(3, "0") // "005"
  • padEnd(targetLength, padString): Pads end of string.
    "5".padEnd(3, "0") // "500"
  • charAt(index): Returns character at index.
    "Hello".charAt(1) // "e"
  • charCodeAt(index): Returns Unicode of character at index.
    "Hello".charCodeAt(1) // 101
  • split(separator): Splits string into array.
    "a,b,c".split(",") // ["a","b","c"]

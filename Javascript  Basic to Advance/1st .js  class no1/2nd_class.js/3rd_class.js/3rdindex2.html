Dialog Boxes

There are three types of Dialog Boxes

  1. alert("Your message goes here.")
  2. confirm("Your message goes here.")
  3. prompt("Your message goes here.")
Note:
Is HTML file mein JavaScript ke dialog boxes (alert, confirm, prompt) ka use dikhaya gaya hai. Agar aap inko test karna chahte hain, toh browser ke Dev Tools (F12) open karein, Console tab mein ye commands run karke inka behavior dekh sakte hain.
  • Dev Tools open karne ke liye F12 ya Ctrl+Shift+I press karein.
  • Console tab mein upar diye gaye code ko paste karein aur Enter press karein.
  • Alert, Confirm, Prompt ka pop-up dekhne ke liye alag alag commands try karein.
Is tarah aap dialog boxes ka use aur unka behavior samajh sakte hain.
